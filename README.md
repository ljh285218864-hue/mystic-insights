# Mystic Insights WordPress Theme

A premium spiritual guidance, tarot, astrology and fortune-telling WordPress theme.

## What is included

- `style.css` with the required WordPress theme header and complete responsive design system.
- `functions.php` with theme setup, menu registration, title support, featured image support and asset loading.
- `index.php` as a complete conversion-focused homepage.
- `assets/js/main.js` for mobile menu behavior and reveal state.

## Install in WordPress

1. Download or clone this repository.
2. Put the `mystic-insights` folder inside `wp-content/themes/`.
3. In WordPress admin, go to Appearance > Themes.
4. Activate `Mystic Insights`.
5. Set your site title, logo and menu from the WordPress Customizer or Site Editor.
